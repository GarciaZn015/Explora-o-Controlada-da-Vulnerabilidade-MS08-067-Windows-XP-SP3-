# Contribuindo

Obrigado por considerar contribuir para este repositório.  
Este projeto documenta estudos práticos em segurança ofensiva e laboratórios controlados.  
Para manter a qualidade e integridade do conteúdo, siga as diretrizes abaixo.

---

## Requisitos Gerais

- Contribuições devem ser feitas **exclusivamente em ambientes legais e controlados**.
- Não envie conteúdo envolvendo exploração de sistemas reais sem autorização.
- Não compartilhe dados sensíveis, credenciais, capturas reais de terceiros ou qualquer material que viole privacidade.

---

## Estrutura das Contribuições

Envie conteúdos no formato abaixo:

1. **Título do experimento**
2. **Objetivo**
3. **Cenário / Ambiente**
4. **Ferramentas utilizadas**
5. **Procedimento**
6. **Resultado**
7. **Aprendizados / limitações**

---

## Como enviar uma contribuição

1. Crie um fork do repositório.
2. Crie uma branch com um nome claro:
3. Adicione seus arquivos seguindo a estrutura do projeto.
4. Abra um Pull Request descrevendo:
- O que foi alterado
- Por que foi alterado
- Evidências (prints, logs, etc.)
5. Aguarde revisão.

---

## Código de Conduta

- Não promova atividade maliciosa.
- Não incentive exploração de sistemas sem consentimento.
- A discussão técnica deve ser objetiva e respeitosa.

---

## Boas Práticas

- Use inglês sempre que possível para aumentar acessibilidade.
- Mantenha o repositório organizado com pastas e nomes consistentes.
- Inclua referências quando usar documentação externa.
- Sempre teste o conteúdo antes de enviar.

---

## Contato

Dúvidas gerais podem ser abertas como *Issues* no repositório.
